# Juice Shaders Lite (free)

3 free 2D shaders for Godot 4, from a solo dev — a taste of the full Juice Shaders pack.

Included here (free, MIT):
- **hit_flash** — flash a sprite to a colour on damage
- **outline** — pixel outline for hover / selection
- **dissolve** — burn a sprite away (or in) with a glowing edge

## Use
Select a Sprite2D → **CanvasItem > Material > New ShaderMaterial** → load a
`.gdshader` from the `shaders/` folder → tweak the parameters. Each shader's header
comment shows how to drive it from code with a Tween.

## Full pack (12 shaders)
Adds CRT, shockwave, water, palette remap, dither, shine sweep, pixelate, wind sway
