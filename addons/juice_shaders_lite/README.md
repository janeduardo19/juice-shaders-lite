# Juice Shaders Lite (Free)

3 free, drop-in 2D shaders for Godot 4 — a taste of the full **Juice Shaders** pack.

Included here (free, MIT):
- `hit_flash` — flash a sprite to a colour on damage
- `outline` — clean pixel outline for hover/selection
- `dissolve` — burn a sprite away/in with a glowing edge

## Install
The files live in `addons/juice_shaders_lite/shaders/`. Select a Sprite2D →
**CanvasItem > Material > New ShaderMaterial** → load a `.gdshader` → tweak the
exposed parameters. Drive them from code with a Tween (see each shader's header
comment).
